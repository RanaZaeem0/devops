# GitHub Pages Deployment Workflow

This project demonstrates Continuous Integration and Continuous Deployment using GitHub Actions.

## Features

- Automatic deployment to GitHub Pages
- Deployment triggered only when `index.html` changes
- Uses GitHub Actions workflow
- Uses GitHub Pages hosting

## Workflow

1. Developer pushes changes to `main`
2. GitHub Actions checks if `index.html` changed
3. Workflow uploads the website artifact
4. GitHub Pages deploys the updated website

## Technologies

- GitHub Actions
- GitHub Pages
- YAML
- HTML
